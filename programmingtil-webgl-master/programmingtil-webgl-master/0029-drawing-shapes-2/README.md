## STARTING OFF:

Make sure you have npm installed.
Run:
```
npm install http-server -g
```

Run:
```
http-server
```

View at: http://localhost:8080/

## CONCEPTS:

* dynamically assigning attributes, uniforms, and varyings to JS objects, in order to dynamically bind them
