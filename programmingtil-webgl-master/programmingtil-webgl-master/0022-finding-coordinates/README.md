## STARTING OFF:

Make sure you have npm installed.
Run:
```
npm install http-server -g
```

Run:
```
http-server
```

View at: http://localhost:8080/

## CONCEPTS:

* Finding Coordinates
* API: gl > vertexAttrib3f
* mouse events: clientX, clientY, target.getBoundingClientRect();
