## STARTING OFF:

Make sure you have npm installed.
Run:
```
npm install http-server -g
```

Run:
```
http-server
```

View at: http://localhost:8080/

## CONCEPTS:

* API > glsl > gl_FragCoord
  * fragment position within the framebuffer using window coordinates.
