## STARTING OFF:

Make sure you have npm installed.
Run:
```
npm install http-server -g
```

Run:
```
http-server
```

View at: http://localhost:8080/

## CONCEPTS:

* API: gl > readPixels
* API: preserveDrawingBuffer
* Type Array Uint8Array(4);
