## STARTING OFF:

Make sure you have npm installed.
Run:
```
npm install http-server -g
```

Run:
```
http-server
```

View at: http://localhost:8080/

## CONCEPTS:

* 3d: ModelView Matrix
  * Combines the Model and View matrices into a single matrix
