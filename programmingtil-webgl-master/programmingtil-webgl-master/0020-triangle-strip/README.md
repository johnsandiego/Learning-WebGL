## STARTING OFF:

Make sure you have npm installed.
Run:
```
npm install http-server -g
```

Run:
```
http-server
```

View at: http://localhost:8080/

## CONCEPTS:

* Drawing triangles with TRIANGLE_STRIP
* API: gl.TRIANGLE_STRIP
