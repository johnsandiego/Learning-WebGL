## STARTING OFF:

Make sure you have npm installed.
Run:
```
npm install http-server -g
```

Run:
```
http-server
```

View at: http://localhost:8080/

## CONCEPTS:

* Drawing multiple points on the canvas object.
* Buffers
* API: bindBuffer, bufferData, vertexAttribPointer, enableVertexAttribArray
